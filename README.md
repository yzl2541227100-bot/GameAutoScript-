# 偃武和平鸽助手

基于 Android 的 SLG 策略游戏自动化脚本引擎，使用 Kotlin + Jetpack Compose 开发。

## 项目简介

偃武和平鸽助手是一款针对偃武次元三国志类 SLG 手游的辅助工具，支持自动集结、采集、日常任务、开荒等功能。采用 OCR 文字识别 + OpenCV 模板匹配 + 百分比坐标三重识别策略，适配不同分辨率设备。

## 技术栈

- 语言: Kotlin
- UI: Jetpack Compose + Material3
- OCR: Google ML Kit 中文文字识别 (bundled)
- 图像匹配: OpenCV 模板匹配
- 截图: MediaProjection (VirtualDisplay)
- 触控模拟: Instrumentation + Accessibility 双模式
- 架构: 多进程 (main / enginfloat / daemon / remote)
- Native: JNI (C++)
