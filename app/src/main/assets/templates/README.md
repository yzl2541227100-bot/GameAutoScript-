# 模板图片目录

将游戏截图中裁剪出的按钮/图标放在此目录下，用于 OpenCV 模板匹配。

## 需要的模板图片:
- war_icon.png - 战争图标 (左侧工具栏)
- join_btn.png - "加入" 按钮
- join_rally_btn.png - "加入集结" 按钮
- back_arrow.png - 返回箭头
- search_icon.png - 搜索图标
- world_map_btn.png - "天下格" 按钮
- main_city_btn.png - "主城" 按钮

## 裁剪要求:
- 从 1280x720 分辨率的截图中裁剪
- 尽量只包含按钮本身，不要太多背景
- PNG 格式，保持透明度
